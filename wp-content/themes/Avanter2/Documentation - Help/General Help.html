Theme Installation

To install your theme in WordPress you must navigate to the menu "Appearance" on the left and then the submenu "Themes". From that page you should see your current theme "Twenty Eleven" by default OR whatever theme you currently have. Now click the "Install Themes" tab and then the text button at the top "Upload". You then should click Choose File and find your theme files on your computer and upload them. Note: You are only uploading the folder "Avanter-WordPress.zip" Your final step is to "Activate" your theme.

If you have any issues installing your theme. Visit this faq: Solving Broken Theme/Stylesheet Missing WordPress Theme Issues

Getting your URLs to display page names instead of numbers

By default the pagenames will be website.com/p15 instead of website.com/about. To fix this go to "Settings" > "Permalinks" and my recommendation is to change the Common Settings to "Day and name". After choosing "Day and Name", Click "Save Changes" and your done. Your urls will now display a proper Permalink structure.


Theme Options & Colors

Color Scheme Modifications

You can easily adjust all the colors of the theme with the theme customizer. This is different from the Theme Options panel. To access the color scheme customizer, you need to be visiting any page of your website. Use the admin bar at the top and hover over your website title and click Customize.

This will open up options for modifying color schemes, navigation and your front page options.



This WordPress theme is driven by the Theme Options located under "Appearance" > "Theme Options". In that section you will find most of the general options for the Theme.

Theme Options - Basic Settings Tab

The Basic Settings tab in the theme options controls the footer widgets and sidebars. Also included is the option to display comments on pages and sidebars on blog pages.

Theme Options - Shop/Portfolio Tab

The shop/portfolio tab in the theme options controls all of the shop and portfolio options and post options. Examples like how many columns per category.

Theme Options - Social Tab

The Social Icons tab in the theme options controls all of the social networking icons that appear in the Social icons widget.

Theme Options - Appearance

The Appearance tab in the theme options controls the Logo and font controls for the theme. In addition the Appearance tab controls the height of the page title.

Theme Options - Tools

The Tools tab in the theme options controls options like basic SEO, Tracking Code, and Custom Code. If you want to make custom additions to the theme, you can do so using the Custom CSS and Javascript section. This is a nice alternative to editing theme files.

Theme Options - Slider Settings

The Slider Settings tab in the theme options controls options for the featured slider.


Navigation Menus

The navigation is easy to customize. You can add pages, categories, or even custom links. Both of the Navigation Menu's are edited in the "Appearance" section under "Menus".

To start, create a Menu Navigation. You just need to enter a "Menu Name" and click "Create Menu".

Create a Menu

For this example, we created a Menu Navigation with the name "Primary Menu".

Select a Menu under Theme Locations

Once you've created the menu, the your next step is to select the Menu as the "Main Navigation" under the "Theme Locations". After you've selected your navigation from the drop-down, click "Save". This is what links your newly created menu to the theme.

Add pages to your Menu

The final step is adding pages and links to your newly created Menu. You can just select a page via the check-box and click "Add to Menu".

Then the page will be added in the Menu list on the right. You can drag menu items around to order them. Or customize/delete the menu items by clicking on the down arrow on the right. Once you are done customizing your menu, click "Save Menu".

Making Sub-Menu's

You can make multiple level's of sub-menus. All you need to do is drag the menu item that you want inside the sub-menu under-neath the parent menu item. There will be an option to indent the item, so that the sub-menu item looks like it is pushed inside of the parent menu item. Widgets

The sidebars are controlled via widgets. You can easily manage the widgets of the theme under Appearance > Widgets.


Drag & Drop your widgets into the corresponding Widget Area

You simply drap & drop the widgets into the spots you want. Drop the widget out of the box if you want to remove it.


Available Widget Areas:

  • Sidebar - Default sidebar available on all pages
  • Shop Sidebar - Default sidebar for shop index and categories
  • Homepage Content Top - Widget area at the top of the homepage page template. Use Progression Home: widgets only.
  • Homepage Content Bottom - Widget area at the top of the homepage page template. Use Progression Home: widgets only.
  • Homepage Footer Widgets - Widget area just above the footer for full width widgets. Use Progression Full Width: widgets only (And twitter widget).
  • Footer - This widget area shows in the footer. You can adjust the column count in the Theme Options > Basic Tab.

Creating the Homepage

By Default the home-page will display the latest blog posts. In order to use the Live Preview homepage, you will need to create a new page that you want to be your homepage. Next, you will go to "Settings" > "Reading" in the WordPress Dashboard.

From there you will change your "Front page displays" from Your latest posts to "A static page". You will want to choose the page you created from the drop-down for 'Front Page'.

Once you've chosen your new Front Page, you can click "Save Changes" and you will now have a custom home-page.

Setting the Home-page Template

The home-page from the Live Preview uses a custom page template named Homepage. You can select this under "Page Attributes" > "Template". This can be done in the right sidebar when editing any page.

Note: You will select this option while editing the actual page.

Homepage Feature Boxes

We built-in some formatting to get your homepage looking like the live preview (Three featured boxes). This is accomplished by using child pages (Each box is a child page of your homepage).

Just create a new page (Each page is one box; so for the live preview we have three child pages) and set it to a child page. You just go under "Page Attributes" and choose the "Homepage" as your Parent page. Then your Page Title and content will get pulled into the homepage.

You can set a featured image if you want OR just include an image in the content area. Shortcodes and images will work just like on any other page.

Note: Adjust the column count of these boxes under the options panel Appearance > Theme Options > Homepage tab.

Second Note: Adjust the order of the pages under "Page Attributes" on the right. There is an "Order" setting.



Regular Homepage Text

You can also add text to your homepage by just typing text in on your homepage. You can build a page using shortcodes and/or images. For example instead of using the homepage boxes above, you can just use the three column shortcodes.



Pull in Featured Area, Shop Products, Blog Posts, or Portfolio Posts (Widgets)

You can pull in Shop Products, Blog Posts, and Portfolio Posts into the homepage. This is done via the the "Homepage Widgets" area under Appearance > Widgets.

Creating the Homepage Featured Slider

The featured slider is automatically created when you select the Homepage template. You can control the slider options under Theme Options > Slider .

Adding slides involves:

  1. Create a Portfolio Post
  2. Assigning that portfolio post to the any Category
  3. Assign your Homepage to the slider Category
Step 1: Create a Portfolio Post

Each slide is a portfolio post. You can use an image by setting a featured image or use a video by using the embed code.

Step 2: Assigning that portfolio post to a category

Next you will assign your portfolio post to a category on the right side of the post page. For our example we will name this 'featured'.

Note: You will have to add the new category the first time. After you have added it, it will appear as a checkbox for any future posts.

Step 3: Assign your Homepage to the slider Category

The last step is assigning your Homepage to the category you created. Then your homepage will pull in all posts from that category. For our example we named this category 'featured' so we will select 'featured'.

Note: Make sure your Slider Category Name doesn't have spaces in it. Remove any spaces from just the category name. All other categories used can have spaces.


Setting an Image for each Slide (Featured Image)

In order to add an image to the slides, you must set the image as a featured image. All of the resizing will be handled automatically. Just upload the image and then click "Use as featured image". See the example below.


Adjusting slider settings

You can adjust the homepage slider settings under the Appearance > Theme Options Panel > Sliders.



Additional Options

Additional slider options can be located below post area under "Post Formats".



Creating Portfolio Pages

Creating the portfolio pages is simple since the theme uses a custom post type of "Portfolio". All you need to do is find the "Portfolio" heading in the sidebar. From there you can click "Add New", which will add a new portfolio item. Or you can start by adding a Portfolio "Category". The Portfolio "Category" is a identifier or group for your portfolio items. This will allow you to have multiple portfolio pages and types.

Here are 4 steps to creating your Portfolio Pages from start to finish:

  1. Add a New "Portfolio Item".
  2. Create a "Category" and add your "Portfolio Item" to that "Category".
  3. Create a Portfolio page and select a "Portfolio Category".
  4. Select a Portfolio Page Template.

Note: Make sure your Portfolio Category Names don't have spaces in them. Remove any spaces from just the portfolio category name. All other names can have spaces. his name does not display anywhere so it will not effect your website.

Step 1: Add a New "Portfolio Item".

Adding a new portfolio item is just a matter of clicking "Add New" under the Portfolio custom post type. To insert your image you just need to Upload an image and then set that image as your Featured Image.

Inserting an Image to your Portfolio Item (Featured image)

Upload an image and set it to featured image. All of the resizing will be handled automatically. Just upload the image and then click "Use as featured image". See the example below.

Adding a video to your portfolio item

You just need to find the embed code to your video and paste that into the Embed Video box. Example:

Note: The video will automatically be resized for you.

<iframe src="http://player.vimeo.com/video/22884674?title=0&byline=0&portrait=0&color=ffffff" 
	width="400" height="225" frameborder="0"></iframe>
Getting the embed code

You can get the video embed code from vimeo, youtube, and etc. easily by clicking on the "Embed" link on their website as shown below in the image:

Step 2: Create a "Category" and add your "Portfolio Item" to that "Category".

You can add a "Category" from the sidebar of your Post as shown in the screen-shot or by clicking Category and adding a new Category. Each Portfolio Item will need to be marked as a "Category" before it can be added to a Portfolio. This will be handled in the sidebar of each Portfolio Post.

Note: Make sure your Portfolio Category Names don't have spaces in them. Remove any spaces from just the portfolio category name. All other names can have spaces.

Step 3: Create a Portfolio page and select a "Portfolio Category".

Once you have created your Portfolio Items and marked them as a "Category". You can then create a page and associated your page with the Category. All you need to do is create a page and then select which Portfolio Category at the bottom under page options.

Step 4: Select a Portfolio Page Template.

The final step to setting up your Portfolio is to select a Portfolio Page Template. This can be done by selecting any of the 4 Portfolio Page templates in the sidebar of your Edit Page area.

WooCommerce is a free plugin that can be found here: http://wordpress.org/extend/plugins/woocommerce/.

After you have completed the installation, you will have a WooCommerce & Products section of your theme.

This theme has some built in short codes to help you build your theme. You can find the short codes demoed in the Live Preview About and Page Styles pages.

Built into the Post Editor

The Short codes are build-in to the post editor so you never have to memorize or look-up a short code again! The second row of buttons in the Post Editor list's out the possible short-codes.

Buttons

The first short code button is "Add a button". The code generated is:

You just need to choose your color and link and that is it Examples:

Sizes: progression-small or progression-medium or progression-large

The "Add a Tabs Pane" short code button generates tabbed content that looks like this:

[tabgroup][tab title="Tab 1"]Tab 1 content here[/tab][tab title="Tab 2"]Tab 2 content here[/tab][tab title="Tab 3"]Tab 3 content here[/tab][/tabgroup]

The "Add a Toggle" short code button generates toggled content that looks like this:

Lightbox

Adding the lightbox functionality requires you to enter the HTML Editor. Once you've Inserted a photo, you will need to click on the HTML tab of the editor.

If you want more control and/or a solution without editing HTML, I suggest this plugin: PrettyPhoto Single Zoom.

<a rel="prettyPhoto" href="http://localhost/wordpress/wp-content/uploads/2011/08/logo-test2.jpg">

Importing Live Preview Files

I have created a folder labeled "Live Preview XML" which contains an export for the Live Preview. The place to upload this XML file is in WordPress under Tools > Import.

Once you've arrived on the Import page, you will click on the "WordPress" link: Wordpress - Install the WordPress importer to import posts, pages, comments, custom fields, categories, and tags from a WordPress export file.

The first time you do this you will get a pop-up prompting you to install the "WordPress Importer". You will need to click "Install Now". Then You will have installed the plugin and need to click "Activate Plugin & Run Importer".

From here you will just need click "Choose File" and select the "freehold.wordpress.2012-07-30.xml" file. Then click "Upload file and import".

You will then be prompted to Assign Authors. You can read the description at the top as this explains everything. Note: Make sure you Import Attachments.

Check the box "Download and import file attachments". This will download attachments for images included in the Live Preview demo.

You now just need to wait for the importer to finish. This might take a few minutes depending on your hosting server speed.

Select the Navigation Menu

After the importer is completed, you will have to select your menu under Appearance > Menus.

Select the Menu as the "Main Navigation" under the "Theme Locations". After you've selected your navigation from the drop-down, click "Save". This is what links your newly created menu to the theme.

Select Your Homepage and Blog

Next you will need to select your homepage and blog under Settings > Reading.

Select your Homepage and Portfolio Categories

Finally, the homepage and portfolio pages will need to be updated. You just need to select the "Slider Category" under Page Options and update the page.

Responsive layout

This site is beautiful at any size, be it a 17" laptop screen or an iPhone. All of the styles for the responsive layout are located here: /css/responsive.css

The responsive grid is controlled via "Media Queries". Magnolia uses media queries to serve the scalable grid, but also for the convenience of styling your site on different size screens. Magnolia's media queries are almost exclusively targeted at max and min widths rather than device sizes or orientations. The advantage of this is browsers and future mobile devices that don't map to exact set dimensions will still benefit from the styles. That being said, all of the queries were written to be optimal on Apple iOS devices. The built in media queries include:

  • Smaller than 960: Smaller than the standard base grid
  • Tablet Portrait: Between 768px and 959px
  • All Mobile Sizes: Less than 767px
  • Just Mobile Landscape: Between 480px and 767px
  • Just Mobile Portrait: Less than 479px

    • Example:

/* #Media Queries
================================================== */

  /* Smaller than standard 960 (devices and browsers) */
  @media only screen and (max-width: 959px) {}

  /* Tablet Portrait size to standard 960 (devices and browsers) */
  @media only screen and (min-width: 768px) and (max-width: 959px) {}

  /* All Mobile Sizes (devices and browser) */
  @media only screen and (max-width: 767px) {}

  /* Mobile Landscape Size to Tablet Portrait (devices and browsers) */
  @media only screen and (min-width: 480px) and (max-width: 767px) {}

  /* Mobile Portrait Size to Mobile Landscape Size (devices and browsers) */
  @media only screen and (max-width: 479px) {}




How to disable the Responsive Ability

You can easily remove the responsive styling by removing the /css/responsive.css file. If you do this, then add a body min-width otherwise tablets will have trouble rendering the background images properly:

body {min-width:1200px;}
